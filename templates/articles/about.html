What is muspy?

Muspy /mjuː'spaɪ/ is a free service that notifies you when your favourite artists release new albums.

How does it work?

Each day muspy checks the MusicBrainz database to see if the artists you follow have new releases, and notifies you if they do. Notifications are sent by email or via RSS.

Who is behind muspy?

Muspy was created and is maintained by Alexander Kojevnikov and Olga Vasylevska. Following hundreds of obscure metal bands we are into is quite a daunting task, so we decided to do something about it. Hopefully you will find muspy useful too.

Which tools do you use?

Muspy is written in Python using the Django framework. We use GNU/Emacs for text editing, Pinta and Inkscape for raster and vector graphics, and Wink for screencasting.

Muspy wouldn't exist without the MusicBrainz database. If you like muspy, consider contributing to the MusicBrainz project.

Source code?

Muspy is free and open source software released unde GNU AGPL. The code is hosted on GitHub.
